# ROADMAP.md

# app_hr 开发路线图（Roadmap）

Version: 1.0

## 项目愿景

打造一个可私有部署、基于 Streamlit + Pandas + OpenAI Compatible API 的
AI 数据分析平台。

------------------------------------------------------------------------

# Phase 1：基础框架

## M1 项目初始化

目标： - 建立目录结构 - 创建虚拟环境 - 配置 requirements.txt -
完成基础配置管理

验收： - 项目可启动 - Streamlit 首页可访问

------------------------------------------------------------------------

## M2 Excel 导入

目标： - 上传 Excel - 支持多 Sheet - DataFrame 缓存

验收： - 能读取常见 Excel 文件 - 支持中文列名

------------------------------------------------------------------------

## M3 DataFrame Profile

目标： 生成： - 行数 - 列数 - 字段类型 - 缺失值统计 - 前 N 行样例

验收： - Profile 可供 Prompt 使用

------------------------------------------------------------------------

# Phase 2：AI 能力

## M4 LLM 接入

目标： - OpenAI SDK - OpenAI Compatible API - 支持配置 Base URL、API
Key、Model

验收： - 能完成一次对话

------------------------------------------------------------------------

## M5 Prompt Builder

目标： - System Prompt - User Prompt - DataFrame Schema Prompt

验收： - Prompt 模块独立维护

------------------------------------------------------------------------

## M6 Python 代码生成

目标： - LLM 输出 Pandas 分析代码 - 返回统一格式

验收： - 简单查询可生成正确代码

------------------------------------------------------------------------

## M7 安全执行器

目标： - AST 校验 - 白名单 - 禁止危险模块

验收： - 阻止危险代码执行

------------------------------------------------------------------------

# Phase 3：分析与展示

## M8 表格结果展示

目标： - DataFrame - 排序 - 筛选 - 下载 Excel

------------------------------------------------------------------------

## M9 图表生成

目标： 支持： - 柱状图 - 折线图 - 饼图 - 散点图 - 箱线图 - 热力图

优先 Plotly。

------------------------------------------------------------------------

## M10 AI 结果解释

目标： 根据 Python 执行结果生成自然语言分析。

------------------------------------------------------------------------

# 每个里程碑开发要求

每个 Milestone 必须包含：

-   功能目标
-   输入
-   输出
-   模块修改范围
-   单元测试
-   验收标准
-   文档更新

禁止一次开发多个 Milestone。
