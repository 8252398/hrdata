# app_hr

AI Excel 数据分析平台（app_hr）项目文档入口。

## 项目简介

app_hr 是一个基于 **Streamlit + Pandas + OpenAI Compatible API**
的企业级 AI 数据分析平台。

设计目标：

-   本地运行、支持私有部署
-   使用自然语言分析数据
-   Python 负责所有计算
-   LLM 负责理解需求、生成分析代码和解释结果
-   模块化、可维护、可扩展
